# 產品型錄網站 - 模板規則

## 檔案命名格式
```
{產品類別}產品型錄_{YYYYMMDD}_{HHMM}.html
```
**範例：** `門用五金產品型錄_20260206_1430.html`

---

## 目錄結構
```
型錄網站/
├── {產品類別}/
│   ├── index.html          ← 產品型錄網頁
│   ├── LINE@_QR.jpg        ← LINE QR Code
│   └── images/             ← 產品照片
│       ├── {型號資料夾}/
│       └── ...
├── 共用資源/               ← 未來擴充
└── CLAUDE.md               ← 本規則文件
```

## 定價同步規則

當用戶說「同步 XX 定價」時，從 Excel 母表讀取牌價並更新網頁。

### 母表位置
```
C:\Users\詹家騏\Desktop\摩瑪建材產品線\摩瑪已下單產品\
├── 門用五金_摩瑪建材/【母表】摩瑪門用五金_完整內部資料.xlsx
├── 電子鎖_摩瑪建材/【母表】摩瑪電子鎖_完整內部資料.xlsx
├── 基礎五金_摩瑪建材/【母表】摩瑪基礎五金_完整內部資料.xlsx
└── 輕奢石_摩瑪建材/【母表】摩瑪輕奢石_完整內部資料.xlsx
```

### 對應欄位

| 產品線 | 工作表 | 品名欄位 | 牌價欄位 | 照片欄位 |
|--------|--------|---------|---------|---------|
| 門用五金 | 門鎖價目表（完整版） | 摩瑪品名 | 牌價 | 產品照片 |
| 電子鎖 | 完整成本表 | 摩瑪型號 | 牌價 | — |
| 輕奢石 | 產品型號對照表 | 摩瑪名稱 | 牌價 | 照片 |

### 照片路徑轉換

母表照片路徑需轉換為網頁相對路徑：
```
母表：/門鎖系列/產品照片/T86-YN，HL/12.png
網頁：images/T86-YN，HL/12.png
```

詳細設定檔：`sync_config.json`

## 正式頁與測試頁同步提醒

### 重要
**修改本目錄（正式頁）的版型或功能後，必須詢問用戶是否同步更新「型錄網站_測試」目錄（測試頁）**

### 需要詢問同步的項目
- Header 結構和樣式
- Hero 區域
- Filter 篩選列
- 產品卡片樣式
- Footer
- Lightbox
- discount-fab 折扣按鈕
- 手機版 RWD 樣式
- **新增功能**
- **JavaScript 邏輯**
- **任何 HTML/CSS/JS 修改**

### 測試頁位置
```
C:\Users\詹家騏\Desktop\website\型錄網站_測試\
├── 門用五金/index.html
├── 電子鎖/index.html
├── 基礎五金/index.html
└── 輕奢石/index.html
```

### 同步注意事項
測試頁有額外的購物車/會員功能，同步時需保留這些功能，詳見測試頁的 CLAUDE.md
